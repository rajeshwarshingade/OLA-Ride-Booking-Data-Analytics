
# 📊 OLA Analytics Dashboard

An interactive, professionally branded **Business Intelligence (BI) dashboard** built with **HTML, CSS, JavaScript, Chart.js, and Leaflet** to analyze and visualize **OLA Ride Booking Data**.  

## 🚀 Features
- **Dynamic Charts & KPIs**: Interactive bar, line, and pie charts powered by Chart.js.  
- **Geospatial Insights**: Ride distribution and hotspot mapping using Leaflet.  
- **Branded UI/UX**: Custom color palette, typography, and logo placement for professional presentation.  
- **Interactive Filters & Slicers**: User-driven analytics with smooth filtering.  
- **Export Options**: Working buttons to export reports in PDF, PPT, and CSV formats.  
- **Responsive Layout**: Optimized for desktop and mobile viewing.  

## 📈 Business Context
This dashboard transforms raw OLA booking data into **actionable insights**:
- Ride volume trends over time  
- Market share breakdowns and competitor comparisons  
- Revenue and performance KPIs  
- Regional demand analysis with maps  

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, JavaScript  
- **Visualization Libraries**: Chart.js, Leaflet  
- **Data Source**: Excel/CSV integration for real-time analytics
